# 📚 Bookstore

Ein selbst entwickeltes Bookstore-Projekt, das während meiner Weiterbildung entstanden ist.  
Die Anwendung dient zur Verwaltung und Darstellung von Büchern und wurde entwickelt, 
um praktische Erfahrungen in der Softwareentwicklung und Anwendungslogik zu sammeln.

---

## 🚀 Funktionen

- Bücher anzeigen und kommentieren
- Übersichtliche Benutzeroberfläche
- Strukturierte Datenverwaltung
- Einfache Navigation innerhalb der Anwendung

---

## 🛠️ Verwendete Technologien

- HTML / CSS / Java
- GitHub

---

## 🎯 Ziel des Projekts

Dieses Projekt wurde entwickelt, um wichtige Grundlagen der Softwareentwicklung praktisch anzuwenden und den Aufbau
einer vollständigen Anwendung besser zu verstehen.

Dabei standen besonders folgende Themen im Fokus:

- Objektorientierte Programmierung
- Datenverwaltung
- Benutzeroberflächen
- Programmstruktur und Architektur
- Fehlerbehandlung und Debugging
- Arbeiten mit GitHub
